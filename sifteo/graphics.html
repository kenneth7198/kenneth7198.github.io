Graphics How To

Quick steps for getting sprites into your game:

  1. Specify a directory for images in your manifest with the “imagePath” parameter.
  2. Put all your images in that directory, in PNG or GIF format.
  3. Open Siftrunner; go to ImageHelper.
  4. Browse to your image directory and name the asset bundle. The bundle can have any name, but it must live in the image directory.
  5. Convert the images.
  6. Quit and restart Siftrunner.
  7. Test your game.

See the Sifteo.Cube reference for more info on the API available to render graphics to the cubes.

Color management

Graphics on Sifteo cubes use an 8-bit color space of format RRRGGGBB. This color space maps to a fixed 256-color palette:

One color is reserved for the transparency key: RGB 72, 255, 170. Pixels of this color will not be painted.

ImageHelper

ImageHelper is a utility within Siftdev used to convert PNG, GIF, and JPG images to a more cube-friendly format.

Access ImageHelper in the developer build of Siftrunner, under the Developer menu.

ImageHelper

To build a sftbndl bundle for your app, browse to your app's image directory and specify the name of the bundle in the Bundle input.

  • The bundle must live in your app's image directory.
  • There must be only one bundle in your app's image directory.

To convert a single image for preview or testing, browse to the individual file, the convert it.

To load a previously converted siftimg file, Select Browse in the Converted panel.

ImageHelper Command Line

You may also generate sftbndl files from the command line:

On Windows: 

Siftdev.exe --create-sftbndl /path/to/images/directory

On Mac: 

Siftdev.app/Contents/MacOS/Siftdev --create-sftbndl /path/to/images/directory

where /path/to/images/directory specifies the location of the directory containing the images you want to bundle. This will create images.sftbndl in the given directory

Performance

To ensure the best performance for sprite strips, be sure to tile your sprites vertically. This allows the cubes to access them in the most efficient manner.
