Search the cube set for a neighborhood that is stacked bottom to top.

Cubes must be neighbored with the same orientation to be considered stacked.

Parameters:
cubesA CubeSet from BaseApp
Returns:
A Cube[] sorted from bottom to top. If there are multiple subsets of neighbored cubes, the result of this function is undefined.
