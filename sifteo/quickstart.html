Quickstart

For the impatient, here's the quickest way to get up and running developing a Sifteo cubes app.

Installation

First, install some software:

  • install mono 2.10.x from http://www.go-mono.com/mono-downloads/download.html (the latest stable version, not the long term support version)
    • on OS X, the mono runtime is sufficient, as opposed to the SDK
  • [recommended] install MonoDevelop from http://monodevelop.com/Download
    • on Windows, you'll also need .NET and GTK#
Note:
while the Mono run-time supports Linux, Siftrunner and the Sifteo SDK are not yet available for this platform.

Generate a New Project

project_gen is a tool that will generate a project skeleton for you, making it easy to get up and running on a new app.

  • Open a command shell and navigate to SIFTEO_SDK/tools/project_gen, where SIFTEO_SDK is the location you installed the Sifteo SDK
    • for example 
      cd /Users/liam/dev/Sifteo/Sifteo_SDK_1_1_2
  • execute 
    mono project_gen MyAppName (-t /path/to/template)
    where MyAppName is the name of your new app, and the path to the template to use is an optional argument
  • this will generate a new directory MyAppName, containing your new app
  • copy your new app wherever you like, open it in MonoDevelop
  • update the References to point at Sifteo.dll within the dependencies folder in your Sifteo SDK installation.

Build and Run Your Project

  • open the .sln file created by project_gen in MonoDevelop
    • on Windows, make sure you're running apps on mono rather than .NET. In Tools -> (Preferences or Options) -> .NET Runtimes, select your mono installation as the default.
  • select Build All from the Build menu
  • select Run from the Run menu - the game should now be waiting for Siftdev to connect
  • open Siftdev, select Load Apps from the Developer menu, and choose the folder that contains your new app
  • your new app should now be included in the app library. click on it, and hit Play to begin running your app
  • that's it!
