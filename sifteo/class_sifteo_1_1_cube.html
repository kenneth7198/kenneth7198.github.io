Detailed Description

Cube represents an instance of a single Sifteo cube.

It provides access to sensor data through its various events, and you can poll for them at any time as well. Cubes are identified by their UniqueId - this remains constant across disconnects and across game sessions. In fact, the Cube instance for a given Cube is also maintained across disconnects as well, so you can be guaranteed that the UserData, and any other state on that instance is still intact.
