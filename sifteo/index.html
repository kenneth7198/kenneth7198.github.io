Welcome to the Sifteo SDK Alpha

Welcome and congrats on getting started with the Sifteo SDK (Software Development Kit) Alpha build.

Within this reference document, you'll find all the info you need to create games and applications for Sifteo cubes. We can't wait to see what you create!

Note:
The Sifteo SDK is still in an alpha state and as such, is subject to change in the future.

Intended Audience

Sifteo cubes are for everyone - inquisitive kids, imaginative adults and innovative makers like yourself. The Sifteo Software Development Kit (SDK) Alpha build is intended to enable designers, hackers, hobbyists, tinkerers and developers of all kinds to build the next great Sifteo game or application.

System Requirements

In addition to a working Mac (10.5+) or Windows PC (XP / Vista / Win 7) with minimum system requirements (see here), you must also have the Mono runtime (http://www.mono-project.com) installed. Mono is an open-source, cross-platform C# and .NET environment that supports OS X and Windows.

If you don't have Sifteo cubes, you can use Siftulator Cheat Sheet, the included cube simulator, to build apps and games

Sifteo SDK Alpha Contents

The Sifteo SDK Alpha contains the following:

  • SiftDev - Siftdev is Siftrunner, the desktop app that provides the runtime for Sifteo cube apps, with additional tools provided for developers
  • Siftulator-00_05_09.jar - the simulator that allows you to test your games without using actual cube hardware
  • /demos - A series of basic projects to help you explore various aspects of the Sifteo cubes and the Sifteo SDK
  • /demo-docs - the source files for the demos above, annotated with explanatory details
  • /api-docs - the reference documentation you're currently reading
  • /Licenses - licenses for the Sifteo SDK and its related libraries & utilities
