Detailed Description

Represents a single audio sample playback instance. Typically, you'll want to create a new Sound from BaseApp.Sounds and then manage its playback state via the returned Sound instance. Volume scale is from 0.0-1.0. Be sure to control the volume of multiple sounds playing at once to avoid clipping the output.
