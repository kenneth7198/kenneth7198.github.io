快速開始指南
第一次使用？
  1. 點擊「教學模式」開始互動式學習
  2. 瀏覽「儀表板」查看即時資料
  3. 在「設備管理」中探索虛擬設備
  4. 嘗試上傳圖片到「媒體庫」
開發者工具
  • 按 F12 開啟瀏覽器開發者工具
  • 查看 Network 標籤觀察 API 請求
  • 使用 Console 執行 JavaScript 代碼
  • 後端 API 文檔: localhost:3001/api-docs
