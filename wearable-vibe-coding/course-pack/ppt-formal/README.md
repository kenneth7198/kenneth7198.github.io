# 正式版錄課 PPT 套件（16 週）

這份套件提供你直接錄製線上課程需要的內容：

- `master-16weeks-ppt.md`：16 週正式 PPT 內容（含作業頁與每週提示詞）
- `recording-script-16weeks.md`：每週錄影講者口白（照念即可）

## 使用方式（VS Code）

1. 安裝 `Marp for VS Code`
2. 開啟 `master-16weeks-ppt.md`
3. 右上角 Marp 預覽
4. 匯出為 `PPTX` 或 `PDF`

## 建議錄課切片

- 每週 3 小時，切成 6 支影片（每支 20~35 分鐘）
- 每支片尾固定：重點回顧 + 作業提醒 + 下集預告
