# 完整課程材料清單（16 週 / 48 小時）

## A. 每位學生硬體包（建議）

1. ESP32 DevKit x1
2. 麵包板 x1
3. 杜邦線（公對公/公對母）x1 組
4. LED（紅/綠/藍）各 5 顆
5. 電阻（220Ω、1kΩ、10kΩ）各 10 顆
6. 按鈕 x4
7. 光敏電阻 x2
8. 電位器 x1
9. 主動蜂鳴器 x1
10. 震動馬達（3V）x1
11. NPN 電晶體（2N2222）x2
12. 二極體（1N4148）x2
13. MPU6050 x1
14. WS2812B / NeoPixel 8 顆環或燈條 x1
15. 心率感測模組（如 MAX30102）x1（可 2 人 1 組）
16. 鋰電池 3.7V + 升降壓/保護板 x1 組
17. USB 線（可傳輸資料）x1
18. 魔鬼氈、束帶、絕緣膠帶、熱縮套管（共用）

## B. 老師示範與備品（1 班）

- ESP32 備品：5~10 塊
- 感測器備品：每類至少 3 組
- 可調電源供應器 x1
- 焊台 + 錫絲 + 吸錫器 x1 組
- 萬用電表 x2
- 第三手工具 x2

## C. 軟體與帳號

- Arduino IDE（2.x）
- ESP32 Board Manager
- 必要函式庫：`Adafruit NeoPixel`、`Wire`、`WiFi`、`HTTPClient`、`esp_now`
- AI 工具帳號：ChatGPT / Claude（二選一即可）
- 協作平台：Google Drive / Notion（繳交 Prompt Log）

## D. 錄課設備（商業上架最低規格）

- 攝影機：1080p/30fps 以上（手機可）
- 收音：領夾麥 + 備援錄音
- 補光：兩盞柔光燈
- 畫面擷取：OBS（螢幕 + 實拍雙機位）
- 剪輯：DaVinci Resolve / Premiere

## E. 法務與授權檢查（上架前）

- 背景音樂為可商用授權
- 投影片圖片來源可商用
- 範例程式碼註記授權
- 學生作品示範需取得同意
- 平台課程條款與退款政策已設定
