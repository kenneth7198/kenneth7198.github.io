# 錄製與上架 SOP（可收費）

## 1) 錄課製作節奏（建議）

- 先錄 Week 1~4 當 MVP（先上架測市場）
- 觀察完課率與問答後，再補 Week 5~8
- 最後上架 Week 9~16 與專題章節

## 2) 每週影片切片（3 小時課）

建議切為 6 支影片，每支 20~35 分鐘：

1. 概念導入（知識點）
2. 硬體接線
3. AI Prompt 示範
4. 程式執行與除錯
5. 作業引導
6. 常見錯誤 QA

## 3) 錄影腳本模板（每支影片）

- 開場 30 秒：本段目標 + 完成成果
- 主內容 15~25 分鐘：一步一步示範
- 除錯段 5~8 分鐘：刻意示範 1 個常見錯誤
- 結尾 1 分鐘：下段預告 + 作業提醒

## 4) 上架前品質檢查

- 音量一致（-14 LUFS 參考）
- 字幕可讀（中英名詞一致）
- 每支影片有對應教材檔案連結
- 範例程式可在乾淨環境跑過一次
- 課後作業有明確評分規準

## 5) 課程商品化必要頁面

- 課程介紹頁（學習成果、適合對象、先備條件）
- 常見問題（要不要會程式？硬體要買哪些？）
- 退款政策（7~14 天）
- 聯絡管道（Email / 社群）
