# 穿戴科技與 AI 協作互動設計（ESP32 × Vibe Coding）完整教材包

此教材包為 16 週 × 每週 3 小時（共 48 小時）設計，可直接用於：

- 大學實體課
- 線上錄播課
- 企業工作坊（精簡版）

## 你會拿到的內容

- `slides/`：16 份可轉 PPT 的講義（Marp Markdown）
- `code/`：16 份每週主範例程式（ESP32 / Arduino IDE）
- `docs/materials-checklist.md`：完整硬體、軟體、拍攝、授權清單
- `docs/recording-and-release-guide.md`：錄課、剪輯、上架 SOP
- `docs/platform-and-pricing-tw.md`：台灣可收款平台與定價方案

## 建議使用方式

1. 每週先用 `slides/weekXX.md` 上課（或錄影）
2. 示範 `code/weekXX_main.ino`
3. 依投影片中的「作業」與「期中/期末里程碑」收作業

## 如何把 slides 匯出為 PPT

推薦安裝 VS Code 擴充：`Marp for VS Code`

1. 開啟 `slides/week01.md`
2. 右鍵 `Open Preview`
3. 執行 Marp Export → `pptx`
