## 電源規劃重點

```
WS2812B 電流估算（最大值）：
  每顆 LED（全白全亮）≈ 60mA
  12 顆燈環 × 60mA = 720mA

教學建議（安全省電）：
  亮度控制在 80/255（31% 亮度）
  → 12 顆燈環實際消耗 ≈ 225mA
  → 一般 5V 2A 電源完全足夠

注意事項：
  ✓ ESP32-C3 和 WS2812B 分開供電（共地）
  ✓ WS2812B 電源端加裝 1000μF 電容
  ✓ 切勿直接用 USB Hub 供 WS2812B（電流不足）
```

## 軟體安裝清單（學員電腦）

請學員**提前安裝**，節省課堂時間：

| 軟體 / 函式庫 | 安裝方式 | 備注 |
|--------------|---------|------|
| Arduino IDE 2.x | 官網下載 | Windows / Mac / Linux |
| ESP32 開發板套件 | 開發板管理員搜尋 ESP32 | 選 3.x 版 |
| **FastLED** | 程式庫管理員 | WS2812B 控制 |
| **NimBLE-Arduino** | 程式庫管理員 | BLE 通訊（Day1 + Day2）|
| **ArduinoJson** | 程式庫管理員 | JSON 解析 |
| **WebSockets** by Markus Sattler | 程式庫管理員 | WebSocket（Day2）|
| nRF Connect | App Store / Google Play | 手機 App |

**Arduino IDE 關鍵設定**：
```
開發板：ESP32C3 Dev Module
USB CDC On Boot：Enabled  ← 重要！否則 Serial 不輸出
燒錄速度：921600
```
